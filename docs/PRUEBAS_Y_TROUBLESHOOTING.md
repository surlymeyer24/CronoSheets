Casos Problemáticos\n\n### Error 1: \"invalid_token\"\n\n```\nOwner Bridge falló: {\"status\":\"error\",\"message\":\"invalid_token\"}\n```\n\n**Causa:**\n- El token en el main NO coincide con el token en el owner\n- Se cambió el token en uno pero no en el otro\n\n**Solución en el OWNER:**\n```javascript\ngenerateAndSetOwnerToken()\n// Copia el token que retorna\n```\n\n**Solución en el MAIN:**\n```javascript\nsetOwnerBridgeConfig(\n  'https://script.google.com/macros/s/[URL_DEL_OWNER]/exec',\n  '[TOKEN_QUE_COPIASTE_ARRIBA]'\n)\n```\n\n### Error 2: \"owner_bridge_http_403\" o \"owner_bridge_http_404\"\n\n```\nOwner Bridge falló: {\"status\":\"error\",\"message\":\"owner_bridge_http_403\"}\n```\n\n**Causa:**\n- La URL no es correcta\n- La URL no está desplegada como Web App\n- Permisos de acceso insuficientes a la URL\n\n**Solución:**\n1. En el proyecto OWNER, ve a **Implementar → Nuevas implementaciones**\n2. Copia la URL de la Web App\n3. En el MAIN:\n```javascript\nsetOwnerBridgeConfig('[URL_CORRECTA]', '[TOKEN]')\n```\n\n### Error 3: \"No se tienen permisos para desproteger\"\n\n```\ndesprotegerHoja: ... como cuenta-b@empresa.com\n  → Intentando vía Owner Bridge...\n  ✗ Owner Bridge falló: {\"status\":\"error\",\"message\":\"...\"}\n  → Intentando desproteción local...\n  ✗ Error: Access denied\n  ⚠ ERROR CRÍTICO: No se tienen permisos para desproteger\n```\n\n**Causa:**\n- Owner Bridge no está funcionando (ver Error 1 o 2)\n- El usuario NO fue agregado como editor de la protección anterior\n\n**Solución:**\n1. **Opción A:** Configura correctamente el Owner Bridge (ver arriba)\n2. **Opción B:** El propietario ejecuta:\n```javascript\naplicarProteccionTotal()\n// Esto recreará todas las protecciones con los usuarios correctos\n```\n\n### Error 4: \"spreadsheetId_requerido\"\n\n```\nOwner Bridge falló: {\"status\":\"error\",\"message\":\"spreadsheetId_requerido\"}\n```\n\n**Causa:**\n- El payload enviado al Owner no incluye el spreadsheetId\n- Error interno en el código\n\n**Solución:**\n- No debería ocurrir después de nuestros fixes\n- Si ocurre: regenera los archivos desde backup\n\n---\n\n## 🔧 Configurar Owner Bridge (si no funciona)\n\n### En el Proyecto OWNER:\n\n```javascript\n// 1.
